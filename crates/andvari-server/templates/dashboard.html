Get started

Most of Andvari is driven from the CLI. The web UI surfaces state and history.

  1. Use the bootstrap token from setup to authenticate the CLI:
    andvari login --token <TOKEN> --workspace {{ workspace_slug }}
  2. Create a project and environment, then write your first secret:
    andvari set DATABASE_URL --project app --env prod
  3. Optionally enable SSO by setting ANDVARI_OIDC_* environment variables and restarting the server.
  4. Browse secrets, tokens, and the audit log from the sidebar.
